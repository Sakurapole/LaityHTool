# LaityHTool

> 基于electron构建的桌面工具集

> 开发文档地址：https://sakurapole.github.io/2021/08/06/laityhtool/

### 目前计划实现的功能
1. Todo 已完成
2. BDownloader 进行中
3. SpyderCenter 
4. NoteReader
5. 图床

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build
```
